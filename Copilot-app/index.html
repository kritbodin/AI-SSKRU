🛠️ Workshop: เรียนรู้ & ทดลองใช้งาน Microsoft Copilot

ตามคำแนะนำในหัวข้อ “3. การใช้งาน Copilot” จากเว็บไซต์ AskMe คู่มือ Microsoft Copilot 2025 :contentReference[oaicite:1]{index=1}

📌 วัตถุประสงค์

  • นักศึกษาทดลองใช้งานฟีเจอร์ Copilot เช่น Chat, Word, Excel, OneNote
  • ฝึกการส่งคำสั่ง (Prompt) และใช้งานจริงผ่านบัญชีมหาวิทยาลัย

🧭 ขั้นตอน Workshop (45–60 นาที)

🔐 ขั้นที่ 1: เข้าถึงระบบ

  1. เปิด Windows 11 แล้วกด Win + C เพื่อเปิด Copilot Chat :contentReference[oaicite:2]{index=2}
  2. ล็อกอินด้วยบัญชีมหาวิทยาลัย (Microsoft 365) ที่มีสิทธิ์ใช้งาน Copilot
  3. ตรวจสอบว่ามีเมนู Copilot ใน Word, Excel, OneNote หรือ %AppName%

💬 ขั้นที่ 2: ทดลองใช้งาน Copilot Chat

  • พิมพ์ Prompt เช่น “สรุปการประชุมล่าสุดใน Teams”
  • ส่งไฟล์หรือภาพถ่าย และลองสั่งให้ Copilot วิเคราะห์หรือสรุป

Tip: ใช้ฟีเจอร์ Vision — ให้ Copilot “ดู” ภาพ/ไฟล์ แล้วช่วยวิเคราะห์ :contentReference[oaicite:3]{index=3}

📝 ขั้นที่ 3: ใช้งานใน Word

  • เปิดเอกสาร Word → แท็บ Copilot → เลือก “Generate Content”
  • ส่ง Prompt เช่น “ช่วยเขียนบทความสรุปหัวข้อ [หัวข้อใดหัวข้อหนึ่ง]”
  • หรือใช้ Copilot แบบ inline เพื่อตรวจไวยากรณ์/ปรับสำนวน :contentReference[oaicite:4]{index=4}

📊 ขั้นที่ 4: ใช้งานใน Excel

  • เปิด Excel → แท็บ Formulas หรือ Data → คลิก Copilot
  • เลือก “Suggest Formula” หรือ “Analyze Data”
  • ลอง Prompt เช่น “คำนวณค่าเฉลี่ยยอดขายจากข้อมูลนี้” หรือ “สร้างกราฟและสรุปผลกำไร-ขาดทุน” :contentReference[oaicite:5]{index=5}

📚 ขั้นที่ 5: ใช้งานใน OneNote

เปิด OneNote ใช้ Copilot เพื่อจัดระเบียบโน้ต จัดกลุ่มหัวข้อ หรือสรุปข้อมูล

ตัวอย่าง Prompt: “สรุปเนื้อหาในหน้านี้เป็นหัวข้อย่อย 5 ข้อ”

🧩 กิจกรรมสรุปผล (10 นาที)

  • ให้นักศึกษาทำสรุปสั้นๆ ว่า “ผมใช้ Copilot ช่วย… และช่วยผมได้อย่างไร”
  • แชร์ปัญหา-ข้อดีที่เจอขณะใช้งานจริง

🎯 ผลลัพธ์ที่คาดหวัง

  • นักศึกษาเรียนรู้การใช้งาน Copilot ในชีวิตเรียนจริง เช่น สรุป เขียน และวิเคราะห์ข้อมูล
  • ได้รับประสบการณ์ใช้งาน AI ช่วยเพิ่มประสิทธิภาพงานผ่านบัญชีจริง

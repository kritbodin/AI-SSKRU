เริ่มเรียนรู้

หลักการ RGC

R - Relevance

ความเกี่ยวข้อง
องค์ประกอบที่สะท้อนเอกลักษณ์และจุดประสงค์ของชมรม รวมถึงสัญลักษณ์ที่เกี่ยวข้องกับสาขาวิชา

G - Goal & Gestalt

เป้าหมายและรูปแบบ
การจัดองค์ประกอบให้สื่อความหมาย สร้างความสวยงาม และเกิดการรับรู้ที่ชัดเจน

C - Clarity & Consistency

ความชัดเจนและสอดคล้อง
การใช้สีและฟอนต์ที่เหมาะสม อ่านง่าย และสามารถใช้งานได้หลากหลายสื่อ

ตัวอย่าง Prompt การออกแบบโลโก้

ชมรมโปรแกรมเมอร์นักศึกษา
Prompt R - Relevance G - Goal & Gestalt C - Clarity & Consistency
เทคโนโลยีและการเขียนโปรแกรม
ออกแบบโลโก้สำหรับชมรมเขียนโปรแกรมที่แสดงความทันสมัยและเทคโนโลยี
• รูปแบบวงจรไฟฟ้าหรือพิกเซล
• สัญลักษณ์ { } ผสานกับอักษร "ก"
• ความเป็นดิจิทัล นวัตกรรม
• เทคโนโลยีการเขียนโปรแกรม
• สร้างความรู้สึกทันสมัย แม่นยำ
• รูปแบบเรขาคณิตเรียบง่าย
• จุดเด่นที่สัญลักษณ์โค้ด
• ความเป็นอนาคต
• สีน้ำเงินไฟฟ้า ฟ้าอ่อน เทาเข้ม
• ฟอนต์แบบ monospace
• ใช้ได้ทั้งพื้นขาว-ดำ
• ข้อความไทย-อังกฤษ
ชมรมศิลปกรรมนักศึกษา
ความคิดสร้างสรรค์และศิลปะ
สร้างโลโก้ที่เต็มไปด้วยความคิดสร้างสรรค์และจินตนาการทางศิลปะ
• จานสีผสานพู่กัน รูปหัวใจ
• เอฟเฟ็กต์สีน้ำกระเซ็น
• เครื่องมือศิลปิน ผ้าใบ
• ความเป็นศิลปะไทยร่วมสมัย
• แสดงความสร้างสรรค์ เร้าใจ
• สีสันสดใส มีชีวิตชีวา
• การแสดงออกอย่างอิสระ
• ความเป็นศิลปิน
• แดงเลือดหมู เหลืองอำพัน เขียวมรกต
• ฟอนต์แบบวาดมือ มีเส้นโค้ง
• พื้นผิวกระดาษหรือผ้าใบ
• มีชีวิตชีวาแต่อ่านง่าย
ชมรมผู้ประกอบการรุ่นเยาว์
ภาวะผู้นำและความสำเร็จ
สร้างโลโก้ที่แสดงภาวะผู้นำและความสำเร็จทางธุรกิจ
• ลูกศรเพิ่มขึ้น ผสานหลอดไฟ/เฟือง
• การจับมือ เครือข่ายธุรกิจ
• ความเจริญเติบโต ความสำเร็จ
• ภาวะผู้นำเยาวชน
• ความเป็นองค์กรและพลวัต
• การเติบโตและนวัตกรรม
• ความมั่นคงและก้าวหน้า
• ความเป็นผู้นำ
• น้ำเงินองค์กร เขียวสำเร็จ ทองเด่น
• ฟอนต์ sans-serif แข็งแกร่ง
• รูปทรงเรขาคณิตสมดุล
• ลวดลายตารางเบา ๆ
ชมรมอนุรักษ์สิ่งแวดล้อม
ธรรมชาติและความยั่งยืน
ออกแบบโลโก้ที่แสดงความรักธรรมชาติและความยั่งยืน
• ต้นไม้งอกจากหนังสือเปิด
• ใบไม้กลายเป็นผีเสื้อ
• โลกหรือธรรมชาติผสมผสาน
• การอนุรักษ์และศึกษา
• เป็นธรรมชาติ ยั่งยืน มีความหวัง
• การศึกษาด้านสิ่งแวดล้อม
• ความกลมกลืนกับธรรมชาติ
• การรับผิดชอบต่อโลก
• เขียวป่า น้ำเงินท้องฟ้า น้ำตาลดิน
• ฟอนต์ธรรมชาติ แต่งใบไม้
• สัญลักษณ์รีไซเคิล หยดน้ำ
• ธรรมชาติแต่อ่านชัด
ชมรมดนตรีนักศึกษา
ความไพเราะและศิลปะเสียง
สร้างโลโก้ที่สื่อความไพเราะและศิลปะแห่งเสียงดนตรี
• กุญแจเสียงไหลเป็นคลื่นเสียง
• โน้ตดนตรีบินเป็นวงกลม
• เครื่องดนตรี ไมโครโฟน
• จังหวะและความไพเราะ
• ความสง่างาม ศิลปะ จังหวะ
• ความไหลลื่นและเคลื่อนไหว
• ความสามัคคีและการ์โมนี่
• ความสร้างสรรค์ทางดนตรี
• ม่วงราชา ส้มสดใส เหลืองอ่อน
• ฟอนต์ไหลลื่น มีจังหวะ
• รูปเงาเครื่องดนตรีเบา ๆ
• สวยงามและอ่านง่าย

คู่มือการใช้งาน

สำหรับนักศึกษา

  • เลือกตัวอย่างที่ตรงกับสาขาที่สนใจ
  • อ่านทั้ง 4 คอลัมน์เพื่อเข้าใจหลัก RGC
  • รวม 4 คอลัมน์เป็น prompt เต็ม
  • ปรับแต่งรายละเอียดตามต้องการ
  • นำไปใช้กับ AI tools เช่น Midjourney, DALL-E

ตัวอย่างการใช้งาน

"ออกแบบโลโก้ชมรมโปรแกรมเมอร์นักศึกษา โดยใช้รูปแบบวงจรไฟฟ้าผสานสัญลักษณ์ { } กับอักษรไทย สร้างความรู้สึกทันสมัยและแม่นยำด้วยรูปทรงเรขาคณิต ใช้สีน้ำเงินไฟฟ้า ฟ้าอ่อน และเทาเข้ม พร้อมฟอนต์แบบ monospace"

เคล็ดลับการปรับแต่ง

  • เปลี่ยนชื่อชมรมให้ตรงกับความเป็นจริง
  • เพิ่มชื่อมหาวิทยาลัยหรือคณะ
  • ระบุขนาดและรูปแบบไฟล์ที่ต้องการ
  • ทดลองหลายรอบจนได้ผลลัพธ์ที่พอใจ
  • ปรับสีตามเอกลักษณ์สถาบัน
